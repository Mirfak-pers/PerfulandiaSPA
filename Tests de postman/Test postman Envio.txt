# Postman Test Cases for EnvioController

## 1. Obtener todos los envíos
- **Método:** GET
- **URL:** http://localhost:8080/api/v1/envios
- **Headers:** Accept: application/json
- **Tests:**
    - Respuesta 200 OK y body es un array (puede estar vacío o tener objetos Envios)
    - Si no hay envíos, respuesta 404 Not Found

---

## 2. Crear/Actualizar un envío
- **Método:** POST
- **URL:** http://localhost:8080/api/v1/envios
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "id": 1,
        "sucursal": { "id": 1, "direccion": "Av. Central", "horaApertura": 9, "horaCierre": 18 },
        "productos": [],
        "estado": "pendiente",
        "origen": "Bodega Central",
        "destino": "Sucursal Norte"
    }
    ```
- **Tests:**
    - Respuesta 201 Created y body contiene el objeto Envios creado/actualizado

---

## 3. Obtener un envío por ID (existente)
- **Método:** GET
- **URL:** http://localhost:8080/api/v1/envios/1
- **Headers:** Accept: application/json
- **Tests:**
    - Respuesta 200 OK y body contiene el objeto Envios con id=1

---

## 4. Obtener un envío por ID (no existente)
- **Método:** GET
- **URL:** http://localhost:8080/api/v1/envios/9999
- **Headers:** Accept: application/json
- **Tests:**
    - Respuesta 404 Not Found

---

## 5. Actualizar un envío existente
- **Método:** PUT
- **URL:** http://localhost:8080/api/v1/envios/1
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "id": 1,
        "sucursal": { "id": 1, "direccion": "Av. Central", "horaApertura": 9, "horaCierre": 18 },
        "productos": [],
        "estado": "enviado",
        "origen": "Bodega Central",
        "destino": "Sucursal Norte"
    }
    ```
- **Tests:**
    - Respuesta 200 OK y body contiene el objeto Envios actualizado

---

## 6. Actualizar un envío no existente
- **Método:** PUT
- **URL:** http://localhost:8080/api/v1/envios/9999
- **Headers:** Content-Type: application/json
- **Body (JSON):** (igual que arriba, pero con id inexistente)
- **Tests:**
    - Respuesta 404 Not Found

---
