# Postman Test Cases for ReporteRendimientoController

## 1. Crear un reporte de rendimiento (válido)
- **Método:** POST
- **URL:** http://localhost:8080/api/v1/ReporteRendimiento
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "idReporte": "REN-001",
        "fecha": "2024-06-01",
        "ventasTotales": 150000,
        "gastosTotales": 50000,
        "utilidadNeta": 100000,
        "observaciones": "Buen mes"
    }
    ```
- **Tests:**
    - Respuesta 201 Created y body contiene el objeto ReporteRendimiento creado

---

## 2. Crear un reporte de rendimiento (inválido)
- **Método:** POST
- **URL:** http://localhost:8080/api/v1/ReporteRendimiento
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "idReporte": "",
        "fecha": "",
        "ventasTotales": -1,
        "gastosTotales": -1,
        "utilidadNeta": -1,
        "observaciones": ""
    }
    ```
- **Tests:**
    - Respuesta 500 Internal Server Error
