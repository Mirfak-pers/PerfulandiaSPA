# Postman Test Cases for SucursalController

## 1. Obtener todas las sucursales
- **Método:** GET
- **URL:** http://localhost:8080/api/v1/Sucursal
- **Headers:** Accept: application/json
- **Tests:**
    - Respuesta 200 OK y body es un array de sucursales (puede estar vacío)
    - Si no hay sucursales, respuesta 204 No Content

---

## 2. Actualizar una sucursal existente
- **Método:** PUT
- **URL:** http://localhost:8080/api/v1/Sucursal/1
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "direccion": "Nueva Dirección 123",
        "horaApertura": 8,
        "horaCierre": 20
    }
    ```
- **Tests:**
    - Respuesta 200 OK y body contiene la sucursal actualizada

---

## 3. Actualizar una sucursal no existente
- **Método:** PUT
- **URL:** http://localhost:8080/api/v1/Sucursal/9999
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "direccion": "Dirección Falsa",
        "horaApertura": 9,
        "horaCierre": 18
    }
    ```
- **Tests:**
    - Respuesta 404 Not Found
