# Postman Test Cases for ReporteVentasController

## 1. Crear un reporte de venta (válido)
- **Método:** POST
- **URL:** http://localhost:8080/api/v1/ReporteVentas
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "idReporte": "VEN-001",
        "fecha": "2024-06-01",
        "totalVentas": 200000,
        "cantidadProductos": 50,
        "observaciones": "Ventas del día exitosas"
    }
    ```
- **Tests:**
    - Respuesta 201 Created y body contiene el objeto ReporteVenta creado

---

## 2. Crear un reporte de venta (inválido)
- **Método:** POST
- **URL:** http://localhost:8080/api/v1/ReporteVentas
- **Headers:** Content-Type: application/json
- **Body (JSON):**
    ```json
    {
        "idReporte": "",
        "fecha": "",
        "totalVentas": -1,
        "cantidadProductos": -1,
        "observaciones": ""
    }
    ```
- **Tests:**
